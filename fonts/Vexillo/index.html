Kreative Vexillo

Kreative Vexillo is a collection of images of flags in various formats, sizes, and aspect ratios, as well as the XML files used to describe them and the computer programs used to generate them.

This page describes the Kreative Vexillo emoji fonts, TrueType fonts that contain the flags as embedded color images. The flags are encoded using four methods: private use code points, zero-width joiner sequences, regional indicator symbols, and tag characters.

Flags are present for all officially assigned ISO 3166-1 alpha-2 country codes as well as many exceptionally reserved, transitionally reserved, and user-assigned codes. Many other flags are included, among them ICS and NATO signal flags, pride flags, and flags for constructed languages.

Kreative Vexillo contains over 500 flags. For comparison, the My'file flag icon set contains 300 flags and the original FamFamFam flag icon set contains 247 flags.

The flag images and the XML files used to describe them are placed in the public domain, free to use for any purpose with no requirement for attribution, except where otherwise noted within the XML file. The source code and binaries used to produce them, as well as any other computer programs in this repository including digital font files, are released under the Mozilla Public License.
